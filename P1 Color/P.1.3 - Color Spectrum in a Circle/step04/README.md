# P.1.3 - Color Spectrum in a Circle - Step 4

## The Code

Using a switch statement which takes in the key pressed, set controls to the number keys which change the segment count of the circle.

```js
switch (key) {
    case '1':
        segmentCount = 360;
        break;
    case '2':
        segmentCount = 45;
        break;
    case '3':
        segmentCount = 24;
        break;
    case '4':
        segmentCount = 12;
        break;
    case '5':
        segmentCount = 6;
        break;
}
```
